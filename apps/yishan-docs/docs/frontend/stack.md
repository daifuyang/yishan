# 技术栈与命令

## 依赖概览

- `@umijs/max`（Umi Max v4）
- `antd`、`@ant-design/pro-components`
- `react` 19、`typescript` 5
- 代码质量：`@biomejs/biome`

详见 `apps/yishan-admin/package.json`。

## 常用命令

- 开发：`npm run dev`
- 构建：`npm run build`
- 预览：`npm run preview`
- 单测：`npm test`
- Lint：`npm run biome:lint`、`npm run tsc`
