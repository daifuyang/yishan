# 多语言与国际化

前端开启国际化：`config/config.ts` 的 `locale` 配置，默认 `zh-CN`，并自动根据浏览器语言切换。

- 文案位置：`src/locales/*`，包含多语言的 `component.ts`、`globalHeader.ts`、`menu.ts`、`pages.ts` 等
- 切换入口：右上角 `SelectLang`
- 主题与布局文案：`config/defaultSettings.ts` 与各语言的 `settingDrawer.ts`

若新增语言，按目录结构新增并在 `locale.default` 设置默认语言。
