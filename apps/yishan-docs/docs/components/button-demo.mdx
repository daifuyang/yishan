## 实际应用示例

### 表单提交按钮

<div className="shadcn-example" style={{display: 'flex', flexDirection: 'column', gap: '16px', maxWidth: '300px', marginBottom: '16px'}}>
  <div>
    <label style={{display: 'block', marginBottom: '4px', fontSize: '14px', color: '#374151'}}>用户名</label>
    <input type="text" placeholder="请输入用户名" style={{width: '100%', padding: '8px', border: '1px solid #d1d5db', borderRadius: '4px'}} />
  </div>
  <Button type="submit" style={{width: '100%'}}>提交表单</Button>
</div>

```tsx
import { useState } from 'react';
import { Button } from '@yishan/shadcn';

function LoginForm() {
  const [loading, setLoading] = useState(false);

  const handleSubmit = async () => {
    setLoading(true);
    // 模拟API调用
    await new Promise(resolve => setTimeout(resolve, 2000));
    setLoading(false);
  };

  return (
    <div className="space-y-4">
      <h3>登录表单</h3>
      <Button 
        onClick={handleSubmit}
        disabled={loading}
        className="w-full"
      >
        {loading ? '登录中...' : '登录'}
      </Button>
    </div>
  );
}
```

## CSS类名说明

按钮组件使用Tailwind CSS类名构建，以下是主要样式类：

### 基础样式类
- `.inline-flex` - 内联弹性布局
- `.items-center` - 垂直居中对齐
- `.justify-center` - 水平居中对齐
- `.rounded-md` - 圆角边框
- `.text-sm` - 小号字体
- `.font-medium` - 中等字重
- `.transition-all` - 所有属性过渡效果

## 属性说明

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| variant | 'default' \| 'destructive' \| 'outline' \| 'secondary' \| 'ghost' \| 'link' | 'default' | 按钮变体 |
| size | 'default' \| 'sm' \| 'lg' | 'default' | 按钮尺寸 |
| disabled | boolean | false | 是否禁用 |
| asChild | boolean | false | 是否渲染为子元素 |
| className | string | - | 自定义样式类 |

## 使用须知

1. 确保已正确安装 `@yishan/shadcn` 包
2. 按钮组件基于 Radix UI 和 Tailwind CSS 构建
3. 支持所有原生 button 元素的属性
