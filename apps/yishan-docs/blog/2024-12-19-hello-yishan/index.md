## 什么是移山？

移山是一个现代化的快速开发平台，旨在帮助开发者和团队以最小的成本构建高质量的企业级应用。我们提供：

- **完整的组件库**：开箱即用的UI组件，支持主题定制
- **管理后台模板**：权限管理、数据可视化、CRUD操作一应俱全
- **最佳实践**：基于实际项目经验的开发模式和架构设计
- **开发工具链**：从项目创建到部署的完整工具支持

## 快速开始

### 1. 创建项目

使用我们的脚手架工具快速创建项目：

```bash
npm create yishan-app@latest my-app
cd my-app
npm install
npm run dev
```

### 2. 选择模板

我们提供了多种模板供您选择：

- **基础模板**：包含基础的项目结构和配置
- **管理后台**：完整的后台管理系统模板
- **移动端**：基于响应式设计的移动端应用模板

### 3. 开始开发

项目创建完成后，您就可以开始开发了：

```bash
# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 运行测试
npm run test
```

## 核心特性

### 组件库

我们的组件库基于现代前端技术栈构建：

- **TypeScript**：完整的类型支持
- **Tailwind CSS**：原子化CSS框架，支持主题定制
- **Radix UI**：无障碍访问的基础组件
- **Framer Motion**：流畅的动画效果

### 管理后台

管理后台模板包含：

- **用户管理**：用户、角色、权限管理
- **数据可视化**：图表、报表、仪表盘
- **CRUD操作**：增删改查的完整解决方案
- **文件管理**：文件上传、预览、下载

### 开发体验

- **热更新**：开发时的实时预览
- **代码提示**：完整的IDE支持
- **文档系统**：详细的API文档和使用指南
- **社区支持**：活跃的开发者社区
